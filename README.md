# DB Manager
Ứng dụng Desktop quản lý cơ sở dữ liệu tốc độ cao (Local), giao diện cực đơn giản và tiêu tốn cực ít tài nguyên.

# Chức năng chính
Quản lý Local DB: Kết nối và thao tác dữ liệu ngay tại máy cá nhân.

Tối ưu tài nguyên: Chạy mượt mà, chiếm cực ít RAM nhờ nhân Rust.

Giao diện tối giản: Tập trung vào trải nghiệm người dùng, không rườm rà.

# Công nghệ sử dụng
Core Framework: Tauri

Backend (System): Rust

Frontend (UI): React + TypeScript

Build Tool: Vite
